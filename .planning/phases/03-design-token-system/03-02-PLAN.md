  <action>
Per D-08 (targeted per-animation, not blanket global rule) and D-09 (specific override), add the following `@media` block immediately after the `@keyframes pulse-op` block at the end of global.css:

```css
@media (prefers-reduced-motion: reduce) {
  .size-calculating {
    animation: none;
  }
}
```

This must appear AFTER the `@keyframes pulse-op { ... }` block. The @keyframes definition itself stays unchanged.

Do NOT add a blanket `* { animation: none !important; }` rule — D-08 explicitly rejects this approach.
Do NOT add any transition overrides — only guard existing animations (there is only one: pulse-op).

Run `npm run typecheck` after the change.
