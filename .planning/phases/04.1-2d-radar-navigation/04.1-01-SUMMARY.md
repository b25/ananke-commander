**One-liner:** RadarPane with d3-hierarchy partition sunburst, streaming folder-size IPC wrapper, breadcrumb drill-down, and ResizeObserver SVG sizing.

## What Was Done

- Installed `d3-hierarchy` (layout math only, not full D3 bundle)
- Added `RadarPaneState` interface to `contracts.ts` extending `PaneStateBase` with `rootPath` and `pathHistory`
- Added `'radar'` to the `PaneType` union
- Added `RadarPaneState` to the `PaneState` discriminated union
- Created `useRadarData` hook: lists directory via `listDir`, wraps streaming `startFolderSize`/`onFolderSizeDone`/`onFolderSizeError` IPC into a single `getFolderSizeOnce()` Promise, caps at 50 children per node
- Created `SunburstChart` SVG component: uses `d3-hierarchy partition` layout, renders arcs as SVG `<path>` elements, `<title>` tooltips, click drill-down, center "Up" button, reduced-motion guard, all colors from CSS design tokens
- Created `RadarPane`: breadcrumb navigation bar, drill-down and up-navigation via `drillDown` callback, `ResizeObserver` for responsive SVG sizing, Enter-key event dispatch for file-browser sync (next plan)
- Created `RadarPane.css` stub with design-token-only rules
- Wired `RadarPane` into `App.tsx` renderer switch (required for TypeScript correctness)
